First workflow: browser to Rasbhari

The extension connects to Rasbhari with a base URL and API key, syncs actions and rules, and then asks first or triggers directly depending on mode.

1
Connect
Set the Rasbhari URL and API key, then test the connection.
2
Sync
Fetch actions and rules from Rasbhari so the extension stays generic.
3
Trigger
Use browser context to trigger Activities, events, or quick-log flows.
Start with a few extension-visible actions

Map browser verbs like save_current_page or capture_selection to the Rasbhari target you actually want.

Install Flow

How to try it right now

  1. Download rasbhari-chrome-extension.zip from this page.
  2. Open chrome://extensions and enable Developer mode.
  3. Unzip the bundle and use Load unpacked to add the extension.
  4. Open the extension’s setup page and save your Rasbhari base URL and API key.
  5. Use the extension against /automation/api/extension/status, /sync, and /execute while the richer client behaviors land next.
